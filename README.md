# Guess the 4 Digit Number GUI Game 🎮

A Python GUI game where the computer generates a random 4-digit number and the player tries to guess it using a graphical interface.

## 📌 Features

- Beautiful GUI window
- Random 4-digit secret number
- Input box for guesses
- Button to check answer
- Shows correct digits count
- Displays total tries
- Reset / Play Again option

## 🛠 Technologies Used

- Python 3
- Tkinter
- random module

## ▶️ How to Run

1. Install Python
2. Save file as `guess_gui.py`
3. Run:

```bash
python guess_gui.py
